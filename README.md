# ArticleWebApp

This web app is created using asp.net core 3.2, In this app there are 3 types of users. User can login & register. Admin user can manage their roles. This role & user managment is done using Identity Framework. Also we have integrated Telerik design library in this web app. We have displayed few componenets using Kendo-UI. Also we have made dynamic menu in this app and that is coming from view component.
